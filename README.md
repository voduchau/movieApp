### Requirements
- [Node](https://nodejs.org/) >= 5.0.0
- [npm](https://npmjs.com) >= 3

### Installation

Clone this repo

```sh
$ git clone https://github.com/voduchau/movieApp.git
$ cd movieapp
$ yarn install or npm install
```
### How to start
```sh
$ react-native run-android
$ react-native run-ios
```
